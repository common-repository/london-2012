== Description ==

The 'London 2012 Olympics' plugin will display  constantly a updating list of medals country wise on whichever page you like just use shortcode[oly_london_2012]. It has also been widgetized for drag-and-drop use on your sidebar.

A webservice is used for display constantly updating list of medals from olympics

The listing can be styled to match your site there is no  CSS box so its follow your current style,easy to fit your current layout .

== Webservice/API==

A webservice is used for  this plugin that is the backbone of this plugin (need of this plugin)
security: highly secure,not any type of damage to your site just read data from other site 




== Installation ==

1. Upload the `london_2012` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Navigate to 'olympics' menu from admin panel 
4. You can either drag the `London 2012` widget straight onto your sidebar, or you can place shortcode [oly_london_2012] into your page where you'd like the events to appear

= How do I uninstall the plugin? =

Simply delete the old `london_2012` folder, and remove the shortcode [oly_london_2012] line from your page file.
